🖥️ No Robot? No Problem!

You don't need a physical Reachy Mini robot to use ReachyClaw!

ReachyClaw works with the Reachy Mini Simulator, a MuJoCo-based physics simulation that runs on your computer. Watch your agent move and express emotions on screen while you talk. 

# Install simulator support
pip install "reachy-mini[mujoco]"

# Start the simulator (opens 3D window)
reachy-mini-daemon --sim

# In another terminal, run ReachyClaw
reachyclaw --gradio

🍎 Mac Users: Use mjpython -m reachy_mini.daemon.app.main --sim instead

How it works

OpenClaw controls everything

  1. 🎤 Robot captures your voice
  2. 📝 OpenAI Realtime transcribes your speech
  3. 🧠 Your message goes to OpenClaw (the real brain)
  4. 🤖 OpenClaw responds with text + action tags like [EMOTION:happy]
  5. 💃 ReachyClaw executes the actions on the robot
  6. 🔊 Clean text goes to TTS — robot speaks while moving

Quick start

Get ReachyClaw running with the simulator

# Clone ReachyClaw
git clone https://github.com/shaunx/reachyclaw
cd reachyclaw

# Create virtual environment
python -m venv .venv
source .venv/bin/activate

# Install ReachyClaw + simulator
pip install -e .
pip install "reachy-mini[mujoco]"

# Configure (edit with your OpenClaw URL and OpenAI key)
cp .env.example .env
nano .env

# Terminal 1: Start simulator
reachy-mini-daemon --sim

# Terminal 2: Run ReachyClaw
reachyclaw --gradio
